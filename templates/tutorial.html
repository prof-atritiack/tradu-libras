Tutorial Completo

Siga estes passos para uma experiência perfeita com o TRADULIBRAS

1

Configuração da Câmera

  • Certifique-se de ter uma webcam conectada
  • Escolha um local com boa iluminação
  • Posicione a câmera na altura dos ombros
  • Mantenha o fundo neutro para melhor detecção
2

Posicionamento das Mãos

  • Mantenha as mãos dentro do campo de visão
  • Use gestos claros e bem definidos
  • Faça cada sinal por 2-3 segundos
  • Evite movimentos muito rápidos
3

Tradução

  • Use ponto (.) para falar o texto
  • Sinais são convertidos em texto
  • Texto é transformado em áudio automaticamente
  • Controle de fala automática disponível
4

Mão Robótica

  • Certifique-se que a conexão foi bem sucedida
  • A mão fará um breve movimento quando conectada
  • Reproduz os sinais detectados em tempo real
  • Feedback visual da tradução
5

Controles e Ajustes

  • Apague letras individualmente se necessário
  • Limpe todo o texto com um clique
  • Use o botão de falar para reprodução manual

Dica Importante

Para melhores resultados, mantenha as mãos bem iluminadas e evite fundos muito movimentados. O sistema funciona melhor com gestos pausados e claros.
